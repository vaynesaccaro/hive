Skills OpenRouter

Consultas a múltiplos modelos de IA — Kimi, GPT-5, Grok, Gemini, DeepSeek e mais

7 skills
Configuração necessária: Defina a variável de ambiente OPENROUTER_API_KEY e execute bash _core/mcp/install.sh. Veja o guia de configuração MCP para detalhes. As respostas dos modelos são sempre entregues brutas — sem edição, sem paráfrase.
SkillComandoO que fazCusto est.
Second Opinion /second-opinion Consulta a um único modelo — resposta bruta, zero edição. Pergunta qual modelo usar se não especificado. $0.001–$0.05
Consensus /consensus Mesmo prompt para N modelos em paralelo. Exibe respostas brutas + síntese de convergências/divergências. Modo --deep opcional para crítica cruzada entre modelos. $0.01–$0.20
Anti-Bubble /anti-bubble Modelo neutro lê copy ou pitch SEM contexto da empresa — reação pura de quem está de fora. Identifica jargões e suposições que insiders não enxergam. $0.005–$0.05
ICP Check /icp-check 2–3 modelos simulam seu ICP lendo um copy, e-mail ou landing page. Triangulação multicultural. Lê foundation/icp-audience.md ou descrição inline. $0.01–$0.15
DoD Check /dod-check Modelo barato valida diff do PR contra critérios de aceite do Linear. Extrai a issue pelo nome da branch. Retorna veredicto PRONTO / AJUSTES / REFAZER. $0.001–$0.01
Cost Compare /cost-compare Mesma tarefa em 3–6 modelos do mais barato ao topo de linha. Matriz de custo real em tokens. Ajuda a encontrar o modelo "barato o suficiente" para workers recorrentes. $0.01–$0.50
Code Review (model) /code-review-model Code review por qualquer modelo OpenRouter. Mesma classificação P1/P2/P3 do /codex-review. Use como segunda opinião ou varredura de segurança. $0.01–$0.10

Skills de Dev

Ethan — Tech Lead · Arquitetura, code review, debugging e documentação

8 skills

Pipeline de Code Review

Execute do mais barato ao mais caro. Pare quando encontrar e corrigir problemas.

SkillComandoCustoO que faz
Gemini Review /gemini-review Gratuito Gemini 2.5 Flash via OAuth CLI — revisão estrutural, classificação P1/P2/P3
Codex Review /codex-review ~$0.40 OpenAI Codex CLI — segurança, lógica, performance. Classificação: P1 bloqueia merge · P2 corrigir antes de deploy · P3 follow-up
Claude Review /claude-review Assinatura Claude Opus headless — revisão arquitetural profunda, gate final antes do merge
Quando usar qual:
Mudança trivial (copy, CSS isolado) → nenhum · Nova feature → /codex-review · Mudança de lógica → + /claude-review · Crítico (auth, cobrança, migration) → os três + aprovação sênior
