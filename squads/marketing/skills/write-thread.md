# /write-thread

Write a Twitter/X or Threads thread. Every tweet must stand alone. Tweet 1 is the most important thing you write.

## When to use

When long-form thinking needs to be packaged for a threaded format. Best for breaking down a complex insight, sharing a process, or making a counterintuitive argument step by step.

## Inputs

- `topic`: what the thread is about
- `angle`: the specific take or argument (not just a topic — a position)
- `thread_length`: number of tweets (default: 8)

## Structure

**Tweet 1 — Hook**
A bold claim or counterintuitive insight that forces the "read more." This is the most important tweet. It must:
- Make a specific, falsifiable, or unexpected claim
- Create an itch the reader needs to scratch
- Work as a standalone statement that could go viral alone

Do not start with "Thread:" or "A thread on X." That's not a hook, it's a table of contents.

**Tweets 2 through [N-1] — Development**
One idea per tweet. Each tweet should:
- Advance the argument one clear step
- End with a micro-hook that makes the next tweet feel necessary
- Be able to stand alone if screenshotted

Do not use filler tweets. If a tweet exists only to transition to the next one, delete it and merge the ideas.

**Tweet [N] — Closing**
Actionable takeaway that lands the argument, followed by one CTA:
- `Follow for more on [specific topic]`
- `Repost if this changed how you think about [X]`
- `Reply with [specific question]` — make it a question the reader can actually answer

---

## Output format

```
THREAD — [topic] — [angle]

Tweet 1 (hook):
[text] [character count]

Tweet 2:
[text] [character count]

...

Tweet [N] (closing + CTA):
[text] [character count]

Thread notes:
- [any notes on timing, visual suggestions, or context for publishing]
```

## Rules

- **"Thread below 👇" is never Tweet 1.** It is the laziest possible hook. Cut it.
- **No "In conclusion."** Threads end with an action or an idea, not a summary.
- **Every tweet must stand alone.** If tweet 4 is incomprehensible without tweets 1–3, it's too dependent. Rewrite it.
- **Tweet 1 deserves 40% of your total effort.** If it doesn't perform, no one reads the rest.
- Character limit per tweet: 280 (Twitter/X standard). Flag any tweet over this limit.
